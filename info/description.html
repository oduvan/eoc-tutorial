This is an intro mission, the purpose of which is to explain how to solve missions on Empire of Code and how to get the most out of solving them.

So this mission is the easiest one. Write a function that will receive 2 numbers as input and it should return the multiplication of these 2 numbers.

Input: Two arguments. Both are int

Output: Int.

Example:

{% if interpreter.slug == "js-node" %} 
multTwo(2, 3) == 6
multTwo(1, 0) == 0
{% else %} 
mult_two(2, 3) == 6
mult_two(1, 0) == 0
{% endif %}

How does it work?:

When you start solving the initial code is always consists of an “empty” function (which you need to fill in as the solution) and asserts under this function. You should pay attention to is that your function should return values, and not to print them. {% if interpreter.slug == "js-node" %} That is, use the return command instead of the console.log function. {% else %} That is, use the return command instead of the print function. Check this short explanation. {% endif %}

The asserts after your function you can use in order to check yourself by pressing the “Run” button. Empire of Code also uses several additional tests in order to check your solution when you click the “Check” button.

If the solution didn’t pass the internal tests, the right panel will display an error message containing 3 items.

  • Fail: - shows how your function was called.
  • Your Result: - shows what it returned.
  • Right Result: - what it should’ve returned.

To solve the task the “empty” function must be replaced with the following code.

{% if interpreter.slug == "js-node" %} 
function multTwo(a, b) {
    return a*b;
}
{% else %} 
def mult_two(a: int, b: int) -> int:
    return a*b
{% endif %}

Try to click “Check” button now.
